Objectives

In this stage, you should:

  • Handle Start Button Click Event:

    • After pressing the start button, the timer should start and show the time.
    • Format the time to display minutes, seconds, and hundredths of seconds (mm:ss:ms).
    • Ensure the timer only starts if it's not already running.

  • Handle Stop Button Click Event:

    • After pressing the stop button, stop the timer.
    • Ensure the timer is stopped only if it's currently running.

  • Resume Timer After Pressing Start:

    • After pressing the start button on the stopped timer, it should continue from where it left off.
